# Gurbani API

Access Sikh scriptures via GurbaniNow API.

## Quick Use

```bash
# Search for a keyword
python3 scripts/gurbani.py search "Waheguru"

# Get Ang (page) from Guru Granth Sahib
python3 scripts/gurbani.py ang 1

# Get specific Bani
python3 scripts/gurbani.py bani "Sukhmani Sahib"
```
