# Vedic Texts API

Access ancient Vedic scriptures and Puranas.

## Quick Use

```bash
# Get Rig Veda hymn
python3 scripts/vedic.py rigveda 1.1

# Get a Purana story
python3 scripts/vedic.py purana "Ramayana" 1

# Search mantras
python3 scripts/vedic.py search "agni"
```
