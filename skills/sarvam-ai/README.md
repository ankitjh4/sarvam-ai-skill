## Sarvam AI

Sarvam AI provides state-of-the-art speech and language AI for Indian languages.

### Features
- Text-to-Speech (TTS) with 30+ voices
- Speech-to-Text (STT) 
- Translation (22 Indian languages)
- Transliteration
- Document Intelligence

### Installation
```bash
pip install sarvamai
```
